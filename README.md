### b) Express (Backend)
Install the packages
1. npm install axios
2. npm install mongoose@6.10.0
3. npm install mongodb
4. npm install react
5. npm install express --save
6. npm install body-parser

Functions
1. Login & register
2. Get all movies
3. Get a movie
4. Update a movie
5. Delete a movie
6. Delete all movie
